
#  Space Missions Exploratory Data Analysis (EDA)

This project presents an Exploratory Data Analysis (EDA) of historical space missions using Python. It was completed as part of the **CodeAlpha Data Analytics Internship**. The analysis explores launch trends, mission success rates, leading space organizations, rocket costs, and geographical patterns through data visualization and statistical analysis.

---

##  Project Overview

The objective of this project is to analyze historical space mission data to uncover meaningful insights about the global space industry. The project includes data cleaning, feature engineering, exploratory data analysis, hypothesis testing, and business recommendations.

---

##  Objectives

- Clean and preprocess the dataset.
- Handle missing values and duplicate records.
- Perform feature engineering.
- Explore launch trends over time.
- Identify the top space organizations.
- Analyze mission success rates.
- Study rocket cost distribution.
- Perform statistical hypothesis testing.
- Present insights through visualizations.

---

##  Dataset

The dataset contains information about historical space missions, including:

- Company Name
- Launch Date
- Rocket Name
- Mission Status
- Rocket Status
- Launch Location
- Rocket Cost
- Country

##  Statistical Analysis

The project also performs statistical tests to validate important findings, including:

- Mission success rate across different companies
- Improvement in mission success over time
- Relationship between rocket cost and mission success

---

##  Key Insights

- A small number of companies dominate global space launches.
- Most space missions have been successful.
- Launch activity has increased significantly over the years.
- A few rocket models account for a large share of missions.
- Space launches are concentrated in a limited number of countries.
- Rocket costs vary widely, with a few extremely expensive missions.
